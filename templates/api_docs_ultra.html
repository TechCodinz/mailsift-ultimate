üöÄ Quick Start

1. Get Your API Key

Generate a free API key to get started:

curl -X POST https://mailsift.com/api/generate-key \
  -H "Content-Type: application/json" \
  -d '{
    "email": "your@email.com",
    "tier": "free"
  }'

2. Make Your First Request

Extract emails from text:

curl -X POST https://mailsift.com/api/v5/extract \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "text": "Contact us at support@mailsift.com or sales@company.com"
  }'

üîê Authentication

All API requests require authentication using your API key. Include it in the Authorization header:

Authorization: Bearer YOUR_API_KEY

üîç Email Extraction

POST /api/v5/extract Extract emails from text content

Request Body

{
  "text": "string (required) - Text content to extract emails from",
  "validate": "boolean (optional) - Validate extracted emails (default: true)",
  "format": "string (optional) - Output format: 'json' or 'csv' (default: 'json')"
}

Example Request

curl -X POST https://mailsift.com/api/v5/extract \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "text": "Contact us at support@mailsift.com or sales@company.com for more info.",
    "validate": true,
    "format": "json"
  }'

Example Response

{
  "status": "success",
  "total_emails": 2,
  "valid_emails": 2,
  "emails": [
    {
      "email": "support@mailsift.com",
      "is_valid": true,
      "provider": "business",
      "confidence": 0.98
    },
    {
      "email": "sales@company.com",
      "is_valid": true,
      "provider": "business",
      "confidence": 0.95
    }
  ],
  "credits_used": 1,
  "credits_remaining": 99
}

üåê Web Scraping

POST /api/v5/ultra-scrape Extract emails from websites

Request Body

{
  "url": "string (required) - URL to scrape emails from",
  "max_pages": "integer (optional) - Maximum pages to scrape (default: 5)",
  "follow_links": "boolean (optional) - Follow internal links (default: true)",
  "validate": "boolean (optional) - Validate extracted emails (default: true)"
}

Example Request

curl -X POST https://mailsift.com/api/v5/ultra-scrape \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "url": "https://example.com/contact",
    "max_pages": 3,
    "follow_links": true,
    "validate": true
  }'

‚úÖ Email Validation

POST /api/v5/validate Validate email addresses

Request Body

{
  "emails": ["array of strings (required) - Email addresses to validate"],
  "check_mx": "boolean (optional) - Check MX records (default: true)",
  "check_smtp": "boolean (optional) - Check SMTP server (default: false)",
  "check_disposable": "boolean (optional) - Check disposable domains (default: true)"
}

Example Request

curl -X POST https://mailsift.com/api/v5/validate \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "emails": [
      "support@mailsift.com",
      "invalid@nonexistent.com",
      "test@10minutemail.com"
    ],
    "check_mx": true,
    "check_smtp": false,
    "check_disposable": true
  }'

üîç Keyword Search

POST /api/v5/ultra-search Search emails by keywords

Request Body

{
  "query": "string (required) - Search query or keywords",
  "industry": "string (optional) - Industry filter",
  "limit": "integer (optional) - Maximum results (default: 100)",
  "fuzzy_match": "boolean (optional) - Enable fuzzy matching (default: true)"
}

Example Request

curl -X POST https://mailsift.com/api/v5/ultra-search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "query": "marketing director",
    "industry": "technology",
    "limit": 50,
    "fuzzy_match": true
  }'
